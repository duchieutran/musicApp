# các kiến thức học được
 - xem lại được mobx, api và route, model
 - dùng screenutil để tuỳ chỉnh kích thước theo tỉ lệ màn hình:

  + sử dụng Material3 : Có thể xem thêm ở GG => Túm cái váy lại là dùng tốt hơn
